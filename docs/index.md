# Workflow Wiki

Workflow Wiki is an operating manual for AI-assisted work. It helps builders choose the right workflow for the current stage of a project, understand the specialist role involved, and follow a checklist until the work is done.

## Start Here

```text
Think -> Plan -> Build -> Review -> Test -> Ship -> Reflect
```

Use this site to answer:

- What workflow should I run now?
- What input do I need?
- What output should exist when done?
- What checklist proves the workflow is complete?
- Which related workflow comes next?
